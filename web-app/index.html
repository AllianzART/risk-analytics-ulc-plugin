Hello Sample

This page provides links for connecting to the Hello sample. The UI Engine is deployed as an Applet embedded in a Web page (using the Java Plug-in). The application is deployed as a servlet. The UI Engine connects to the application using HTTP as the underlying transport protocol.

For information on the Java Plug-in, please refer to the Java Plug-in Product Page.

Applet Starts the Hello sample as an applet.

Application Functionality

The Hello World application is a common starting point for developers when learning the ULC technology. This application has no additional functionality beside displaying a label containing the 'Hello World' text. It can also serve as a simple ULC installation verification.
